Gameplay
📋 Description
Functions like classic Flappy Bird — navigate through pipe gaps, score a point per pair cleared, and survive as long as possible. The game runs infinitely until you lose, then shows a Game Over screen with your high score saved and a Play Again option.
🎯 Target User
Casual Gamers
Recreational players with time to kill — controls are self-explanatory from the start.

InputAction
SPACEFlap / Jump
Mouse ClickButton interaction
✨ Planned Features
  • Different bird types with unique speeds and flap distances
  • Pipes with two entry points — harder gap scores bonus points
  • Dynamic backgrounds that change as you progress
🗂 Program Structure
FileRole
FlappyBird.pyMain file — game loop, event handling, collision detection, scoring
Bird.pyBird class — position, gravity, boost mechanics
Pipe.pyPipe class — procedural generation, movement, scoring zones
Background.pyBackground class — scrolling parallax background
Button.pyButton class — interactive UI buttons (Start, Play Again, Quit)
Settings.pyGlobal constants — screen size, speed, colors, fonts
🖼 Assets
FileUsage
fieldBG.jpegScrolling gameplay background
FlappyGameOver.jpegGame over screen overlay
⚡ Challenges
The most challenging part has been getting the game to loop properly after the end screen — making sure state resets cleanly for a new round.
▶ How to Run
# Install dependency
pip install pygame

# Launch the game
python FlappyBird.py

# Requires Python 3.x and Pygame
